# Nolanimations "open-source' WIP maps
This repository contains my unfinished Work In Progress Beat Saber maps that I probably won't finish in the foreseeable future. To give them a final wave of purpose, I will be leaving maps I don't end up finishing here, open for anyone to do whatever with them. Be that adding your own sections and uploading them as a collab, using them as inspiration for your own maps, "borrow" patterns, timing etc., You are free to download any of the zip files I upload here and go ham with them.

## Important
If you do end up uploading parts of these WIPs to Beatsaver, please credit me in the description and be sure to link this repository.
You are also free to invite me as a collaborator, but I may not always accept.
